WinSock

In computing, the Windows Sockets API (WSA), later shortened to Winsock, is an application programming interface (API) that defines how Windows network application software should access network services, especially TCP/IP. It defines a standard interface between a Windows TCP/IP client application (such as an FTP client or a web browser) and the underlying TCP/IP protocol stack. Дескрипторы (Тэги) логического форматирования

  • <strong> (boald) — - жирный шрифт;
  • <em> (Emphasize) — - обозначить особым образом;
  • <strike> (Strike) — - Перечеркнутый;
  • <pre> (Preformated) — - Предварительное форматирование;
  • <span> () — - ни чего не делает ;
  • <strong> (bold) — жирный шрифт с логическим акцентом (важность);
  • <em> (emphasis) — курсивный шрифт с логическим акцентом (выделение);
  • <mark> (выделение) — подсвечивает текст, как маркером (важное или найденное);
  • <abbr> (HTML) — сокращение с пояснением при наведении (атрибут title обязателен);
  • <cite> (Wikipedia) — название книги, статьи, фильма, источника цитаты и т.д.;
  • <code> (main()) — обозначение фрагмента программного кода (моноширинный шрифт);
  • <kbd> (Ctrl + S) — клавиши, которые нужно нажать на клавиатуре (input from user);
  • <samp> (Error: File not found) — пример вывода из программы (sample output);
  • <var> (x + y) — обозначение переменной (используется в формулах, коде);
  • <dfn> (HTML) — термин, который впервые вводится или определяется в тексте;
  • <time> (29 июня 2025) — дата или время в семантически понятном формате (для поиска, календарей, скриптов);
